# 8. robots.txt Prompt

Generate a robots.txt file that:
- Allows all user agents to crawl the entire site.
- Specifies the sitemap location:

```
User-agent: *
Allow: /

Sitemap: https://gabriel-smith-alabama-attorney.com/sitemap.xml
```
